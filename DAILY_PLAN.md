# Mastering Java Daily Plan

This is the working schedule for the repo. Codex adds one project per day at 9:00 PM Eastern.

## Rules

- Add exactly one project per scheduled run.
- Keep all code and README text original.
- Do not copy course prompts, official solutions, hidden tests, or lesson text.
- Compile the new project before committing.
- Use JavaFX only for GUI projects.
- Do not use Swing.
- Update `README.md` so the new day is marked as added and the next day is marked as next.
- Commit and push to `main`.
