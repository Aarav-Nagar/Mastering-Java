## How to Compile

Each project has its own `src` folder. From a project folder:

```bash
javac -d out src/*.java
java -cp out MainClassName
```

For example:

```bash
cd Java/01-gradebook-gpa-calculator
javac -d out src/*.java
java -cp out GradebookApp
```
