# Decision Systems

Small console app that practices building reusable decision logic with clear explanations.

This is practice for:
- `if`/`else` chains, nested conditionals, and compound boolean expressions
- `switch` for choosing a decision system to run
- designing decision functions that return both an outcome and the reasons

## How to Run

From this project folder:

```bash
javac -d out src/*.java
java -cp out DecisionSystemsApp
```
